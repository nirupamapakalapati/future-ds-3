## Objective
The objective of Task 3 is to analyze student feedback data and evaluate overall student satisfaction with respect to teaching quality, course structure, and learning experience.

## Dataset Description
The dataset contains feedback collected from students in the form of ratings (on a numerical scale).  
The key columns in the dataset include:
- Student ID
- Well-versed with the subject
- Explains concepts in an understandable way
- Use of presentation
- Degree of difficulty of assignments
- Solves doubts willingly
- Structuring of the course
- Provides support for students going above and beyond
- Course recommendation based on relevance

## Tool Used
- Power BI Desktop

## Analysis Performed
The following analyses were performed using Power BI:
- Average rating analysis for all feedback parameters
- Distribution analysis of assignment difficulty using a pie chart
- Comparison of feedback parameters using bar and area charts
- Trend visualization using a line chart
- Tabular representation of average scores using a matrix visual

## Key Insights
- Faculty knowledge and clarity of explanation received high average ratings.
- Students were satisfied with presentation quality and doubt-solving support.
- Assignment difficulty was mostly rated as moderate.
- A high number of students recommended the course, indicating strong relevance.

## Conclusion
The overall feedback analysis shows that students are satisfied with the teaching quality and course structure. The Power BI dashboard provides clear visual insights that can help institutions identify strengths and areas for improvement.
